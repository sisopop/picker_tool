# 선택 도구 (Vite + React)

룰렛과 사다리타기를 한 페이지에서 테스트할 수 있는 개인용 웹앱입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

빌드 결과물은 `dist` 폴더에 생성됩니다.

## 배포

### Vercel
1. GitHub에 업로드
2. Vercel에서 저장소 연결
3. Deploy

### GitHub Pages
1. `npm install`
2. `npm run build`
3. `dist` 폴더를 정적 호스팅하거나 Pages용으로 업로드
